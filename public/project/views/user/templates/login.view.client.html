Tutorial Hub

This is a web application built using MEAN stack which provides a learning platform that look for tutorials for the user including both videos and slides.

It collects the tutorials from YouTube and SlideShare for a LEARNER, and lets the TUTOR to upload the video lectures directly on YouTube. It allows user to mark any tutorial as 'Favorite' to look back later.
